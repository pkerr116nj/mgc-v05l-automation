# ATP Companion GC Asia+US Production-Track Constitution

## Why This Package Is Being Advanced

This package is being advanced because the repo evidence now supports a specific production-track wrapper around the GC Asia+US candidate:
- strong standalone economics
- a narrow `US_LATE` safeguard with low collateral damage
- an outer halt-only governance layer that reduces residual ugliness without pretending to fix the strategy

It is being advanced as a production-track candidate package, not as a benchmark promotion.

## What It Is Allowed To Do

- run as an admitted paper package on the shared paper runtime
- use the explicit lane-local `US_LATE` core-probe invalidation safeguard
- use outer halt-only governance at `-3000`
- use the shared operator-control vocabulary
- remain symbol-scoped to `GC`
- remain provenance-explicit and lane-explicit

## What It Is Not Allowed To Do

- replace the frozen ATP benchmark
- imply repo-wide ATP rule changes
- silently broaden to other symbols
- silently broaden to live broker automation
- silently add new promotion/add logic
- silently convert the outer governance wrapper into benchmark semantics

## Rollback / Demotion Triggers

Suspend or demote the package if any of these occur:
- realized paper performance factor degrades more than `25%` versus the admitted-package full-history expectation over a materially informative sample
- realized paper expectancy degrades more than `30%` versus the admitted-package expectation over a materially informative sample
- live-paper divergence exceeds `15%` on trade count or `20%` on realized net P/L over the review window without an explicitly diagnosed cause
- `3` consecutive halted sessions occur
- `2` consecutive sessions halt and still continue deteriorating after the halt threshold was reached
- reconciliation becomes repeatedly dirty or operator truth becomes ambiguous
- the lane experiences a package-local fault that is not clearly infrastructural and recoverable

## What Counts As Success

Success evidence:
- clean shared-paper-runtime operation
- controls target the package correctly
- lane-local artifacts stay coherent
- realized paper remains directionally aligned with the admitted-package expectation
- halt-only governance remains an outer constitution rather than the dominant source of economics

## What Counts As Failure

Failure evidence:
- package behavior cannot be distinguished clearly from raw-candidate or benchmark surfaces
- repeated operator ambiguity around halt / resume / flatten behavior
- realized paper only looks acceptable because governance is amputating too much value
- residual drawdown remains operationally intolerable even under the admitted package

## Promotion Rule

This constitution advances the package only into production-track paper status.

Benchmark replacement still requires explicit promotion.
